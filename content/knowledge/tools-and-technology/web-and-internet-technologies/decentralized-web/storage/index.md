## Overview

Decentralized storage systems replace centralized servers with distributed networks where data is addressed by its content hash rather than its location. This enables censorship resistance, redundancy, and user-owned data persistence.

## Key Technologies

- [[ipfs|IPFS]] — InterPlanetary File System, a peer-to-peer protocol for content-addressed storage and sharing
- [[arweave|Arweave]] — Permanent, decentralized storage using a novel blockweave data structure
- [[ceramic-network|Ceramic Network]] — Decentralized data streaming network for mutable documents on top of IPFS
