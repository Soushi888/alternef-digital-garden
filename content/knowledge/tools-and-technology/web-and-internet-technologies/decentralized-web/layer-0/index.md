## Overview

Layer 0 refers to the foundational infrastructure beneath individual blockchain networks. These protocols enable cross-chain communication, message passing, and interoperability — solving the problem of isolated blockchains that cannot natively communicate with each other.

## Key Protocols

- [[layer-0|Layer 0 Networks]] — Overview of cross-chain infrastructure protocols
- [[inter-blockchain-communication-protocol|Inter-Blockchain Communication Protocol (IBC)]] — Cosmos ecosystem's protocol for blockchain interoperability
- [[mixin-network|Mixin Network]] — A layer-0 network for inter-chain asset transfer and high-speed transactions
- [[nostr|Nostr]] — Simple, censorship-resistant communication protocol using cryptographic keys
