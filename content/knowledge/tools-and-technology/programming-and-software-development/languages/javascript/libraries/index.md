## Overview

JavaScript's ecosystem includes thousands of libraries that extend the language's built-in capabilities. This section covers notable libraries used for DOM manipulation, drag-and-drop interactions, audio playback, rich text editing, data visualization, and other UI concerns.

These libraries complement core JavaScript and framework knowledge documented in [[javascript|JavaScript]], [[svelte|Svelte]], and [[react|React]].

## Libraries in This Collection

### UI and Interaction
- [[sortable-js|SortableJS]] — Touch-friendly drag-and-drop list reordering
- [[muuri|Muuri]] — Responsive grid layouts with drag-and-drop
- [[swiperjs|SwiperJS]] — Mobile-first touch slider and carousel

### Rich Text and Editors
- [[tinymce|TinyMCE]] — Mature WYSIWYG editor with plugin ecosystem
- [[editor.js|Editor.js]] — Block-based rich text editor

### Media and Files
- [[howlerjs|HowlerJS]] — Web audio library for games and apps
- [[filepond|FilePond]] — Flexible file upload with preview

### Animation and Visualization
- [[three-js|Three.js]] — 3D graphics and WebGL rendering
- [[reveal.js|Reveal.js]] — HTML-based presentation framework

### Utilities
- [[js-cookie|js-cookie]] — Simple cookie read/write API
- [[leafletjs|Leaflet.js]] — Lightweight interactive maps
- [[auth-js|Auth.js]] — Authentication for web applications
