# Recent Changes

All entries are sorted by last modification date and updated automatically on each build.

## How to read this page

Three tabs let you explore the garden's activity from different angles:

**All**: Every note sorted by most recent activity. New additions show "added", revised notes show "edited".

**Timeline**: Every note sorted by the date it was first added to the garden, newest first. Use this to discover what has been published recently.

**Recently Edited**: Only notes that have been revised after their initial publication. A note qualifies as "edited" when the gap between its first and latest commit exceeds one hour.

The active tab is remembered across visits. Use "Load more" to paginate within each view.
